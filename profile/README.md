# AresOS ROM

Welcome to AresOS.

A custom ROM project based on **LineageOS 23.2 / crDroid (Android 16 QPR2)**, featuring essential, user-focused enhancements. Originally developed for **POCO X4 GT / Redmi Note 11T Pro (+) / Redmi K50i (xaga - MT6895)**, but it’s now available for all devices to build!

We currently support Android 16 QPR2, with the latest security patches from Google. Status: March 2026
